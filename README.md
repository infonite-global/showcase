## 🏷️ The Name

**INFONITE** = _Information_ + _Infinite_

The brand represents the limitless potential of data extraction and process automation.

---

## 🎨 Color Palette

### Primary Green

- **HEX:** `#00a76f`
- **RGB:** `0, 167, 111`
- **Usage:** Primary accent, CTAs, highlights

### Dark Brand

- **HEX:** `#1C252E`
- **RGB:** `28, 37, 46`
- **Usage:** Dark backgrounds, text

### Pure White

- **HEX:** `#fcfcfc`
- **RGB:** `252, 252, 252`
- **Usage:** Light backgrounds, contrast
